Insight Key Stage Data Source Preferences

Select the data source underpinning your local authority and school values for each key stage of Insight using the controls below.

Data sources automatically roll forward as the year progresses, but if you wish to override an automatically selected data source you can do so here.

A change in data source will be reflected in Insight the following day, once the next overnight caching process is complete.

Once a data source reaches the "NPD (final)" dataset it becomes locked and can no longer be overridden.
